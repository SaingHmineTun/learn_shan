# Learn Shan

A Flutter application designed to help learn and practice the Shan language script.

## Features

- Shan script character learning
- Interactive tables for practicing Shan characters
- Organized collection of Shan language characters grouped by categories


## Project Structure

```
lib/
├── main.dart          # Main entry point of the application
├── screen/
│   ├── home_screen.dart  # Home screen of the app
│   └── table_screen.dart # Screen for displaying character tables
├── utilities/
│   └── utils.dart     # Utility functions and constants (Shan characters data)
└── widget/
    └── list_item.dart # Reusable UI components
```

## About the Shan Language

The Shan language is a Tai language spoken by the Shan people in Myanmar, Thailand, and China. This application helps learners familiarize themselves with the Shan script and practice reading and writing.
